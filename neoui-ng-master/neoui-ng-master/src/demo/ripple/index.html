Andorid L ripple effect

Supports both jQuery an angularjs

In the angularjs just add 's-ripple' attribute to any DOM element, lol :)

Here's how you use it in angularJS, just add 's-ripple' attribute

<div s-ripple duration="800" speed="800"></div>

angular .module( "demo.ripple", [ "$ui.ripple" ] ) .controller( "rippleController", [ "$scope", function( $scope ) { /** Not needed javascript */ } ] );


Attributes

=controller

null

Controller of the 'ripple', See the jQuery method

@speed

0

单位毫秒, 周期值, 大于0循环显示

@duration

300

Value of the 'animation-duration' property

=color

null

为空时候, 设置随机颜色

@auto-bind

true

默认绑定事件到点击事件

@random

false

循环时, 位置随机出现

Use it in jQuery

$( selector ).ripple( options )

<div class="md-ripple"></div>

require( [ "ui/ripple/ripple" ], function() { var ripple = $( ".md-ripple" ).ripple(); } );

Options

speed

0

单位毫秒, 周期值, 大于0循环显示

duration

300

Value of the 'animation-duration' property

color

null

为空时候, 设置随机颜色

autoBind

true

默认绑定事件到点击事件

random

false

循环播放时, 位置随机出现

class4progress

"md-ripple-progress"

循环时宿主元素样式

class4fail

"md-ripple-fail"

调用 hide( status ) 方法时, status 为 false 时添加样式

class4done

"md-ripple-done"

调用 hide( status ) 方法时, status 不为 false 时添加样式

Method

.enabled()
If present, enabled ripple effect
.disabled()
Disable ripple effect
.show()
Show the ink rippe effect
.hide( [status] )

status

Boolean, 默认 true

Work well with 'options.class4done' and 'options.class4fail'
