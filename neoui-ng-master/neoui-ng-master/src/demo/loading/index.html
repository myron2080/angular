A Simple Div Loading

Loading animation with keyframe for LESS & HTML

Use it in angularJS

<s-loading class="md-loading-global" controller="loading" ng-click="loading.hide()"></s-loading>

angular .module( "demo.loading", [ "$ui.loading" ] ) .controller( "loadingController", [ "$scope", function( $scope ) { /** Not needed javascript */ } ] );


Attributes

=controller

null

Controller of the 'loading', See the jQuery method

@auto-show

false

显示 loading 框



Classes

.md-loading-global

全局显示

Use it in jQuery

$( selector ).loading();

<div class="md-loading"></div>

require( [ "ui/loading/loading" ], function() { var loading = $( ".md-loading" ).loading(); } );

Method

loading.show()
显示 loading 框
loading.hide()
隐藏 loading 框
