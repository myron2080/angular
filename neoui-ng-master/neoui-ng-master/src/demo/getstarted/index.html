Deploy

如何部署项目到本地

安装git

What is Git?


安装NodeJs

Get the Node.js


安装Gulp.js

工程构建基于Gulp.js

$ npm install -g gulp

安装Bower.js

JS依赖管理

$ npm install -g bower

安装Less

$ npm install -g less

获取代码

打开项目主页 neoui-ng 点击Download下载项目到本地

或者git clone https://github.com/trazyn/neoui.js.git克隆代码到本地


安装项目构建依赖

$ npm install


安装JS依赖

$ bower install


运行项目

$ gulp

Build

gulp www:dev

启动开发环境http服务


gulp www:dist

启动部署环境http服务


gulp clean

移除build目录


gulp dist:vendor

压缩合并项目依赖JS到dist/vendor.js / vendor.min.js


gulp dist:neoui

构建项目JS(dist/neoui.js)和CSS(dist/neoui.css)


gulp dist:app

构建应用程序


gulp dist

顺序执行dist:vendor, dist:neoui, dist:app


gulp css:app

编译应用样式


gulp css:neoui

输出dis/neoui.css/dist/neoui.min.css


gulp watch

监听服务


gulp data

构造测试数据


gulp jshint

执行代码检测


gulp

顺序执行watch, data, dist, www:dev
